# Breakout - TC2005B

Juego estilo Breakout desarrollado para la tarea de TC2005B.

## Controles

- **Flecha Izquierda / Derecha**: Mover el paddle
- **Espacio**: Servir la pelota

## Objetivo

Destruir todos los bloques con la pelota sin perder las 3 vidas.

## Reglas

- La pelota rebota en los bordes izquierdo, superior y derecho
- Si la pelota pasa por el borde inferior, se pierde una vida
- Al destruir todos los bloques, el nivel avanza y la velocidad de la pelota aumenta
- El juego continúa indefinidamente hasta que el jugador pierde todas las vidas

## Cómo ejecutar

1. Abrir el archivo `index.html` en un navegador web
2. Configurar el número de renglones y columnas si se desea
3. Presionar el botón "Reiniciar" para iniciar el juego
